# Student HTML starter pack
## By phamminhteky1
This is a small project by me to show the examples to use HTML for my friends
# How 2 use
## If you got Github (highly recommended)
1. Fork or clone this repo
2. Go to either index.html or styles.css file and modify it as you like
3. Configure Github Pages at your Settings panel
4. Done!
## If you haven't got Github
1. Download everything in this repo as a zip
2. Unzip the zip file then modify either index.html or styles.css file as you like
3. Plubish your project using any hosting service you like. In some cases, you have to re-zip the file pack.
4. Done!
